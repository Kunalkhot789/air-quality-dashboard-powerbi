## Overview
This project presents an interactive **Air Quality Dashboard** built using **Power BI** to analyze air pollution levels across different locations and time periods.  
The dashboard helps identify AQI trends, high-pollution areas, and pollutant behavior using real-world air quality data.

## Objectives
- Analyze air quality using AQI metrics
- Identify cities/regions with high pollution levels
- Visualize trends over time
- Enable data-driven environmental insights

## Tools & Technologies
- Power BI
- Microsoft Excel
- Data Visualization
- Data Analysis

## Key Insights
- AQI trends across time
- Comparison of air quality across locations
- Pollutant-wise contribution analysis
- Identification of high-risk pollution zones

## Dashboard Features
- Interactive filters and slicers
- Trend charts and KPI cards
- Clean and intuitive visual design
- User-friendly layout for presentations
